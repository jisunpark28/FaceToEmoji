# 8-week growth experiment checklist (one page)

**Product:** [GetFaceToEmoji](https://www.getfacetoemoji.com/)  
**Positioning (mobile):** Upload → auto face blur → Edit → Download. No install, no server upload.  
**Time budget:** 3–5 hours/week (adjust if needed).  
**Start date:** ___________

---

## North Star & guardrails

| Metric | Definition | Week 8 “continue” | Week 8 “pause / pivot” |
|--------|------------|-------------------|-------------------------|
| **North Star** | Sessions that complete upload → blur/detect → download | ≥ **3–5 / week** (not only you) | **0–1 / week** for 8 weeks |
| **Leading** | Unique visitors (Vercel Analytics) | Trending up vs week 1 | Flat ~only you |
| **Quality** | Mobile completion rate (rough: downloads / uploads) | Feels usable on your phone | Frequent crash / reset |
| **Effort** | Distribution experiments logged | ≥ **2 / week** | Code-only weeks |

---

## Weekly rhythm (repeat × 8)

| Day | Task | Done |
|-----|------|------|
| Mon | Pick **one channel** + **one message** for the week (below) | ☐ |
| Tue–Wed | Publish 1–2 assets (Reel / post / comment / link) | ☐ |
| Thu | Check Analytics + note North Star in log | ☐ |
| Fri | 1 product fix **only if** it blocks completion (crash, download, blur) | ☐ |
| Sun | Fill week row in log (30 sec) | ☐ |

**Rule:** No new features unless they unblock upload → blur → download.

## Week 8 decision

| Outcome | Action |
|---------|--------|
| **Continue** | North Star ≥ 3–5/week OR clear upward trend; keep mobile blur positioning |
| **Pivot message** | Some traffic but wrong audience → switch A/B landing (parents vs SNS) |
| **Pause** | 0–1 North Star, no experiments logged → stop or 4-week hiatus |
| **Kill** | No traffic + no enjoyment + no learning → archive |

---

## Do not count as success

- Only you using the site  
- Page views with 0 download  
- Vanity followers without link clicks  
- SEO position for `face to emoji` only (brand is OK)
